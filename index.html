Services

Samuel Magondu Njenga is a Kenyan software engineer currently working for mSurvey . Born sometime in the 80's, I came through the ranks at the University of Nairobi where I studied computer Science. I graduated in 2012 with a Upper Second Class Honors degree. I have been with mSurvey since August 2012. During that time, I have done projects in Kenya, Trinidad and Tobago, Mauritius and Uganda.

As a software developer, I have currently specialised in python however I have varying experiences with these languages; Java, Java for Android, PHP, C. While am not busy cranking out code, I like to read my bible,watch football (An Arsenal Fan), Rugby (Kenya 7s fan) and blog.

Python

I have been coding in python extensively since 2012. I have a wide experience working in django as well as raw python to deploy large scale systems.

Web

I have worked on various python and php web projects using frameworks such as bootstrap, wordpress, joomla and old-fashion html and bootstrap.

System Architectures

I have deployed systems on aws and designed their achitecture to ensute their scalability. I've configured kannel, mysql, apache among others.
